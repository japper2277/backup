"All Filters" Button Color Comparison

Option 1: Current Dark Style

Same dark background as filter pills (current implementation)

Pros: Maintains consistency with selected pills
Cons: Doesn't distinguish between main action and filter toggles

Option 2: Airbnb White Style

White background with gray border (true Airbnb style)

Pros: Clear visual hierarchy, matches Airbnb pattern, distinct from filter pills
Cons: Might not fit with your dark theme aesthetic

Option 3: Light Gray Style

Light gray background with dark text (compromise option)

Pros: Distinct from pills, softer than white, maintains some consistency
Cons: Less dramatic distinction than white option

Design Recommendation

Based on UX best practices and the Airbnb pattern, Option 2 (White Style) is recommended. It creates the clearest visual hierarchy and helps users understand that the "All Filters" button serves a different function than the individual filter pills.
